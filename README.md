# How to Run

This project includes four main files as per **Milestone 3** requirements: one which is the **main project**, one for running **all unit tests**,  one for running **demonstrations**, and one which includes the **writeup**. 

**Note:** Milestone 3 requirements are in 'Milestone_3' folder

## Run All Unit Tests (In 'Milestone_3' Folder)

```bash
python test_milestone3.py
```

## Run All Demonstrations (In 'Milestone_2' Folder)

```bash
python demo_milestone3.py
```

**Note:** All demo output will appear in the txt file "demo_output.txt"
